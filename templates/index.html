Which are the Happiest Countries in 2017?

The project visualizes global happiness state using the 2017 World Happiness Report.

Intro

The dataset behind the Google Fusion Tables is the 2017 World Happiness Report, which measures the state of happiness in the world today. The bar chart shows the top countries with top hapiness scores, and the map visualizes the global happiness state by colour. The dataset is downloaded from Kaggle .
